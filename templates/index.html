How

Depsite my foundation in electrical engineering where I took lower-level programming classes such as Verilog, I built this with the Django web framework. During my time securing my EE degree, I had many lower level programming classes before my university offered "Python for Electrical Engineers" my last year (finally). Hence, my pure software background started with Python (and why this website is built with Django, for starters).

From there, this site is leveraging a free bootstrap template to make everything pretty.
